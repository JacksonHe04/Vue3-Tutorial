08-Other-APIs: Vue 3 其他 API

目录结构

1. Shallow Ref 和 Shallow Reactive

介绍 Vue 3 中的 shallowRef 和 shallowReactive，用于创建浅层响应式引用和对象。

2. Readonly 和 Shallow Readonly

探讨如何使用 Vue 3 中的 readonly 和 shallowReadonly 创建只读数据。

3. toRaw 和 markRaw

介绍 toRaw 和 markRaw，用于控制 Vue 响应式系统中的对象行为。

4. 自定义 Ref

介绍如何使用 Vue 3 提供的 customRef API 自定义响应式引用。
